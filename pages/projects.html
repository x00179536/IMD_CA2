Description

There has been many instances where I have been waiting on a bus to arrive, and it would either cancel, or not arrive at the destination at all, and I thought that this is enough. With the use of the Travel For Ireland (TFI) Developer API, I am now storing data on all of the bus journeys that are being cancelled, and displaying them in a way users can understand so that it is clear to people which bus routes are the most likely to be cancelled based on previously collected data, so they can avoid being disappointed by a bus not showing up.

Use Case

Users may use this tool in order to dictate which route they are taking, being given data and proabilities calculated by data retrieved from the Travel For Ireland (TFI) Developer API.

View collected data that is held within the MySQL DB hosted in GCP.
